SVG with D3

SVG stands for Scalable Vector Graphics, it is a language that allows to build shapes by describing geometrical attributes with code. D3 just makes controlling (create, select and modify) SVG graphics much simpler as it would be without it.


You can draw SVG shapes directly in the HTML file inside the <svg> tags, or use D3. Both options are shown in the code to the right, and the results are seen below. The purple circles, drawn without D3 can still be modified with D3, by refering to their classname 'target'. In the exmple I've added a colored stroke to the circles.


The two red rectangles on the other hand were created with D3 in the <script> section by appending an SVG to the div container called 'chart'. 

                
<body>
    <!-- Draw the svg shapes (circles) directly in the html -->             
    <svg width="100" height="100">
        <circle class="target" style="fill: coral" stroke="black" cx=60 cy=60 r=30></circle>
        <circle class="target" style="fill: cornflowerblue" stroke="black" cx=150 cy=60 r=30></circle>
    </svg>

    <div id="chart"></div>

    <script>
        d3.selectAll(".target").transition().duration(1000).attr("stroke", "darkgreen").attr("stroke-width", 5);

        data = [30, 60]
        d3.select("#chart").append("svg").attr("width", 100).attr("height", 100)
            .selectAll("rect").data(data).join("rect")
            .attr("x", (d, i) => i * 50).attr("y", d => 100 - d)
            .attr("width", d => d).attr("height", d => d)
            .attr("fill", "crimson");
    </script>

</body>
