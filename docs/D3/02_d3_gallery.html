D3 within HTML

Assuming we want to create a chart to be shown on a webpage, then we'll use D3 inside an HTML file.


To get D3 into the HTML you could install D3 in your existing development environment or download the UMD bundles of D3 from the official website and import it in your HTML. In the code to the right D3 is imported via the CDN link, which is the the easiest method to get started.


What you can also see in the code, is the div element with the id="chart", that is the html element we refrenced in the first line of the D3 script one slide before:

var svg = d3.select("#chart").append("svg")

Where we appended an SVG to the div element which will serve as a canvas to draw our chart on.

                
        <html lang="en">
            <head>
                <!-- Load d3.js via CDN link-->
                <script src="https://d3js.org/d3.v7.min.js"></script>
            </head>
        
            <body>

                <!-- Define the svg area where the D3 chart will be drawn -->
                <div id="chart"></div>
        

                <script>
                    
HERE GOES YOUR D3 CODE

                    var svg = d3.select("#chart").append("svg") ...
                </script>

            </body>
        </html>
